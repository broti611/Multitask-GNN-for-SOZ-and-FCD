# Multitask-GNN-for-SOZ-and-FCD

This repository contains the implementation of the multitask Graph Neural Network (MT-GNN) proposed in:

**"Explainable Graph Neural Networks for Joint Seizure Onset Zone and Epilepsy Etiology Classification from Intracranial EEG"**

## Overview

The proposed MT-GNN jointly performs:

- **Node-level classification:** Seizure Onset Zone (SOZ) localization
- **Graph-level classification:** Focal Cortical Dysplasia (FCD) classification

The framework integrates multiple electrophysiological biomarkers represented as patient-specific graphs and incorporates explainability analysis using SHAP visualization.
